## How to Contribute

### Reporting Issues

- Use [GitHub Issues](https://github.com/CYBERNETIC-BIZ/CyberneticAgents/issues) to report bugs or request features
- Include your Python version, OS, and steps to reproduce

### Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Test your changes locally (`cybernetic.biz` to launch the interactive menu)
5. Submit a pull request with a clear description of the changes

### Code Style

- Python 3.10+ compatible
- Follow existing code patterns and conventions
- Use type hints where appropriate
- Keep imports organized (stdlib, third-party, local)

### What We're Looking For

- Bug fixes
- New LLM provider integrations
- Data vendor integrations
- Terminal UI improvements
- Documentation improvements
- Test coverage
